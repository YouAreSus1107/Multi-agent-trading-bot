Live Trading Outputs
data/execution_memory.jsonCritical state file. Pending entries, ATR trailing stops, fills, entry prices. Persists across cycles — do not delete while bot is running.
data/cycle_memory.jsonPosition theses, trade history, cycle summaries. Rebuilt each cycle but accumulates across the session.
data/fundamental_profiles.jsonDaily T-1 universe ranking (top 15 tickers by alpha/beta/vol/momentum). Cached per calendar day — regenerates at midnight.
data/research_log.jsonlAppend-only log of all Bull vs Bear LLM debates and Moderator verdicts. Every cycle appends new entries.
data/reports/*.mdEnd-of-day markdown assessment reports. Each file is timestamped. Read by the premarket briefing at 9:10 ET next day.
data/news_analyst_log.jsonlNews sentiment scores per ticker per cycle.
data/seen_headlines.jsonDedup cache for news items — prevents re-scoring the same headline.
Backtest Outputs

backtest/outputs/Per-run output directory. Contains regime analysis text and PCA visualization chart.
backtest/regime_pca_chart.htmlInteractive PCA chart of HMM regime states. Open in browser to visualize regime clusters.
Optimizer Outputs

backtest/v2_optimization_results.csvAll trials with composite score, per-period metrics, and all 8 parameters. CSV is written live as trials complete — safe to read while optimizer runs.
backtest/test_params.txtBest params in legacy 2-line CSV format. Written by --export-best or pick_best.py. Read by run_backtest_v2.py as default param file.
